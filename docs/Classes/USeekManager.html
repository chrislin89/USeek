USeekManager

@interface USeekManager : NSObject

This singleton class provides the following features

  • Set / Retrieve publisher ID
  • Request server for the points of certain user based on game id
  • publisherId

    Mutable property to set / get publisher id.

    Warning

    You should set publisher id before loading video.

    You can set publisher id in AppDelegate like this.

      ```
  [[USeekManager sharedManager] setPublisherId: @"publisher id"];
  ```

Request
  • -requestPointsWithGameId:UserId:Success:Failure:

    Queries the points user has gained while playing the game. The centralized server will return user’s points based on gameId and userId.

    Precondition

    Publisher ID should be set.

    Declaration

    Objective-C

    - (void)requestPointsWithGameId:(NSString *)gameId
                         UserId:(NSString *)userId
                        Success:(void (^)(int))success
                        Failure:(void (^)(NSError *))failure;

    Swift

    func requestPoints(withGameId gameId: String!, userId: String!, success: ((Int32) -> Void)!, failure: ((Error?) -> Void)!)

    Parameters

    userId

    user’s unique id registered in USeek
    gameId

    unique game id provided by USeek
    success

    block which will be triggered after response is successfully retrieved
    failure

    block which will be triggered when there is an error detected
