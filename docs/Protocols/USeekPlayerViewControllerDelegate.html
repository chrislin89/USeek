USeekPlayerViewControllerDelegate

@protocol USeekPlayerViewControllerDelegate <NSObject>

The USeekPlayerViewControllerDelegate protocol provides a mechanism for your application to take action on events that occur in the USeekWebView. You can make use of these calls by assigning an object to the USeekPlayerViewController’s delegate property directly.
